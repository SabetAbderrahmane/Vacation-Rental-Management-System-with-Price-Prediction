Project Overview

Academic MVP, Project Overview

This project demonstrates the core flow of a vacation rental platform: administrators monitor system data, hosts manage properties, customers book stays, and the system includes a basic ML module for rental price prediction Please note that this is a simplified academic demonstration and not a UI showcase.

Core Features

What the system actually does

Authentication

Registration and login for different user types.

Role-Based Access

Separate flows for administrators, hosts, and customers.

Property Management

Hosts can create, update, and manage vacation rental listings.

Booking Management

Customers can book stays while hosts manage booking confirmations.

Review System

Customers can submit reviews after stays.

ML Price Prediction

Hosts can estimate suitable nightly prices using a trained Scikit-learn model.

Run Locally

How to run the full application

git clone git@github.com:SabetAbderrahmane/Vacation-Rental-Management-System-with-Price-Prediction.git
cd Vacation-Rental-Management-System-with-Price-Prediction

python -m venv .venv
.\.venv\Scripts\Activate.ps1

pip install -r requirements.txt
python init_db.py
python seed.py
python app.py

The full app runs locally at http://127.0.0.1:5000. GitHub Pages only hosts this static showcase page.
